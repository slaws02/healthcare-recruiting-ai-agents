# Recruiting Agent E — Passive Candidate Persona Builder (Demo)

## Purpose

The Passive Candidate Persona Builder generates structured sourcing personas for healthcare roles to support passive candidate targeting and outreach strategy.

It converts role requirements into realistic candidate background patterns, motivators, objections, sourcing channels, and outreach angles.

This agent supports upstream recruiting strategy — before sourcing and outreach begin.

---

## Demonstrates

- Persona modeling for passive candidates
- Motivator and objection prediction
- Background pattern mapping
- Credential pattern identification
- Outreach hook development
- Channel selection guidance
- Tone and engagement strategy

---

## Key Behaviors

- Builds generalized candidate personas (not individuals)
- Avoids naming real employers or people
- Asks clarifying questions when role input is incomplete
- Produces structured sourcing personas
- Supports passive outreach strategy

## Value Pattern

---

## Data Safety Note

This demo agent produces generalized persona models only and does not reference real individuals, employers, or proprietary candidate data.
